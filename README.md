This repository contains all data and code needed to reproduce the figures and tables in the paper:

Frémont *et al.*, *Viral impacts on plankton standing stocks, primary productivity, and biogeochemistry in a model ocean*.

### Download
`git clone https://github.com/PaulFremont3/vDarwin_productivity/`

### Tutorial
The folder `Install_vDarwin/` contains a tutorial on how to install vDarwin. The folder `Simulations/` contain 1 subfolder per simulation that contain parameterisation files for each simulations of the study. The folder `Post_processing/` contains scripts to post process model outputs data and create figure panels from the manuscript. Each folder contains its own tutorial.

Subfolders `Simulations/` and `Post_processing/` contains a README.md that:

1. Describes relevant files
2. Explain how to run the codes
3. Explain where the figure panels are in the generated pdfs (for `Post_processing/`)

Note: The code generates additional outputs beyond those presented in the paper.
